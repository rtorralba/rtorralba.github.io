Os presentamos una guía para la gente que habla español y está viviendo en el país bávaro.

Podéis encontrar toda clase de servicios (abogados, médicos, autoescuelas…) en los que os podréis comunicar en español. También te muestra los servicios que se encuentran cerca de ti usando tu geoposicionamiento, y otras secciones cómo noticias, ofertas de trabajo…

<!-- more -->De momento está disponible para android pero al estar desarrollada con 

<a href="http://ionicframework.com/" target="_blank">ionic framework</a> pronto lo estará para otras plataformas móviles cómo iOS windows phone…

Gran experiencia con este framework para apache cordova que usa angularjs, desarrollo estructurado, buen funcionamiento y aporta muchas herramienta para facilitar el desarrollo de aplicaciones mixtas para dispositivos móviles, permitiendo desarrollar el código una vez y después compilar en las distintas plataformas, android, iOS, windows phone…

Os dejamos también el enlace a Google play para descarga:

<a href="https://play.google.com/store/apps/details?id=com.destinoalemania.alemaniaapp" target="_blank">Descargar de Google play</a>.
